# Day 2 – Normalization

## Learning Points

- Learned the concept of Normalization in databases.
- Understood why Normalization is important in database design.
- Learned that Normalization helps to reduce duplicate data.
- Understood the purpose of 1NF, 2NF and 3NF.
- Learned how Normalization improves data consistency.
- Understood that proper Normalization makes database maintenance easier.
- Learned that good database design avoids insertion, update and deletion anomalies.
- Practiced identifying the normal form for simple tables.

## My Understanding

Normalization is useful for organizing data in a better way. It reduces redundancy and helps in maintaining accurate and consistent data.
